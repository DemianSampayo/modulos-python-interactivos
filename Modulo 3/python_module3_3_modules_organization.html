Importar módulos y organizar código en archivos

A medida que tus programas crecen, organizar el código se vuelve crucial. Python te permite dividir tu código en archivos separados llamados **módulos**, y luego importar y reutilizar las funcionalidades de esos módulos. Esto fomenta la reutilización y mejora la legibilidad.

1. Importación Básica (`import modulo`)

La forma más común de importar un módulo es usando la sentencia `import`. Esto hace que todas las funciones, clases y variables definidas en ese módulo estén disponibles para su uso, prefijándolas con el nombre del módulo.

2. Importación Específica (`from modulo import funcion`)

Si solo necesitas una o unas pocas funciones o clases de un módulo, puedes importarlas directamente usando `from ... import ...`. Esto te permite usarlas sin el prefijo del módulo.

3. Organización de Código en Múltiples Archivos (Simulado)

En proyectos reales, divides tu código en varios archivos (`.py`) para mantenerlo organizado. Un archivo puede contener funciones relacionadas, y otro archivo principal puede importarlas y usarlas.

Simulación de Archivos:

`operaciones.py`:

def sumar(a, b):
    return a + b

def restar(a, b):
    return a - b

`main.py`:

import operaciones

num1 = 10
num2 = 5

suma = operaciones.sumar(num1, num2)
resta = operaciones.restar(num1, num2)

print(f"Suma: {suma}")
print(f"Resta: {resta}")
