Tipos de Datos en Python

Python maneja varios tipos de datos para almacenar diferentes tipos de información. Comprenderlos es fundamental para manipular y procesar datos correctamente en tus programas.

Números Enteros (`int`)

Representan números enteros, positivos o negativos, sin componentes fraccionarios. Son ideales para contar o para valores discretos.

edad = 30
cantidad = -5
año = 2025

Números Flotantes (`float`)

Representan números reales, con parte decimal. Se utilizan para mediciones, cálculos científicos o cualquier valor que requiera precisión decimal.

pi = 3.14159
temperatura = 25.5
precio = 99.99

Cadenas de Texto (`str`)

Son secuencias de caracteres (letras, números, símbolos), encerradas entre comillas simples (`'`) o dobles (`"`). Se usan para texto, nombres, mensajes.

nombre = "Juan Pérez"
mensaje = 'Hola a todos'
codigo = "ABC123"

Booleanos (`bool`)

Representan uno de dos valores: `True` (verdadero) o `False` (falso). Son fundamentales para la lógica condicional y las decisiones en los programas.

es_activo = True
tiene_permiso = False
es_mayor_edad = (edad >= 18)

Listas (`list`)

Colecciones ordenadas y mutables de elementos. Pueden contener diferentes tipos de datos. Se definen con corchetes `[]`.

frutas = ["manzana", "banana", "cereza"]
numeros = [1, 2, 3, 4]
mixta = [1, "texto", True]

Tuplas (`tuple`)

Colecciones ordenadas e inmutables de elementos. Una vez creadas, no se pueden modificar. Se definen con paréntesis `()`.

coordenadas = (10.0, 20.5)
colores_rgb = (255, 0, 0)
dias_semana = ("Lun", "Mar", "Mié")

Conjuntos (`set`)

Colecciones desordenadas de elementos únicos. Son mutables, pero sus elementos deben ser inmutables. Útiles para operaciones matemáticas de conjuntos.

numeros_unicos = {1, 2, 3, 3, 4} # -> {1, 2, 3, 4}
letras = {'a', 'b', 'c'}

Diccionarios (`dict`)

Colecciones desordenadas de pares clave-valor. Cada clave debe ser única e inmutable. Son perfectos para almacenar datos estructurados.

persona = {
    "nombre": "Ana",
    "edad": 28,
    "ciudad": "Madrid"
}
