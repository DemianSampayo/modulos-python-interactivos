Instalar Python y VS Code

Para empezar a programar en Python, necesitarás instalar el intérprete de Python y un editor de código. Visual Studio Code (VS Code) es una excelente opción. Sigue estos pasos para configurar tu entorno de desarrollo según tu sistema operativo.

Instalación en Windows

Paso 1: Instalar Python

Descarga el instalador de Python para Windows desde el sitio web oficial: python.org/downloads/windows. Durante la instalación, **es crucial marcar la opción "Add Python to PATH"** para que puedas ejecutar Python desde la línea de comandos.

Paso 2: Instalar Visual Studio Code

Descarga e instala VS Code para Windows desde su sitio web oficial: code.visualstudio.com/download. Sigue las instrucciones del instalador.

Paso 3: Instalar la Extensión de Python en VS Code

Abre VS Code. Ve a la vista de Extensiones (Ctrl+Shift+X). En la barra de búsqueda, escribe "Python" y busca la extensión publicada por **Microsoft**. Haz clic en "Install".

Instalación en macOS

Paso 1: Instalar Python

macOS suele venir con una versión antigua de Python. Es mejor instalar una versión más reciente. Descarga el instalador de Python para macOS desde: python.org/downloads/mac-os. Sigue los pasos del instalador. Esto instalará Python en una ubicación separada y no interferirá con la versión del sistema.

Paso 2: Instalar Visual Studio Code

Descarga e instala VS Code para macOS desde su sitio web oficial: code.visualstudio.com/download. Arrastra la aplicación a tu carpeta de Aplicaciones.

Paso 3: Instalar la Extensión de Python en VS Code

Abre VS Code. Ve a la vista de Extensiones (Cmd+Shift+X). En la barra de búsqueda, escribe "Python" y busca la extensión publicada por **Microsoft**. Haz clic en "Install".

Instalación en Linux

Paso 1: Instalar Python

La mayoría de las distribuciones de Linux vienen con Python preinstalado. Sin embargo, para obtener la última versión o gestionar múltiples versiones, se recomienda usar `pyenv` o instalarlo a través del gestor de paquetes de tu distribución (ej. `sudo apt install python3` para Debian/Ubuntu o `sudo dnf install python3` para Fedora/RHEL).

# Ejemplo para Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

Paso 2: Instalar Visual Studio Code

Descarga el paquete `.deb` o `.rpm` de VS Code para tu distribución Linux desde: code.visualstudio.com/download. Alternativamente, puedes seguir las instrucciones para instalarlo a través del repositorio oficial de VS Code.

# Ejemplo para Ubuntu/Debian (usando apt)
sudo apt update
sudo apt install software-properties-common apt-transport-https wget
wget -q https://packages.microsoft.com/keys/microsoft.asc -O- | sudo apt-key add -
sudo add-apt-repository "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main"
sudo apt update
sudo apt install code

Paso 3: Instalar la Extensión de Python en VS Code

Abre VS Code. Ve a la vista de Extensiones (Ctrl+Shift+X). En la barra de búsqueda, escribe "Python" y busca la extensión publicada por **Microsoft**. Haz clic en "Install".

¡Con estos pasos, tu entorno de desarrollo Python estará listo en el sistema operativo de tu elección!
