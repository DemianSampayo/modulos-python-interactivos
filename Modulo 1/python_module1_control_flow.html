Control de flujo: `if`, `else`, `elif`

El control de flujo es la capacidad de tu programa para tomar decisiones y ejecutar diferentes bloques de código basándose en ciertas condiciones. Las sentencias `if`, `elif` (else if) y `else` son las herramientas principales para esto.

1. `if`, `elif`, `else` Básico

La estructura `if-elif-else` permite al programa evaluar múltiples condiciones en orden. La primera condición que sea verdadera se ejecuta, y el resto se ignora. Si ninguna es verdadera, se ejecuta el bloque `else`.

Puedes colocar una sentencia `if` dentro de otra sentencia `if` para manejar condiciones más complejas y específicas. Esto permite tomar decisiones basadas en múltiples criterios.
