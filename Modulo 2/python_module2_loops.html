Repetir tareas con `for` y `while`

Los bucles son estructuras de control que te permiten ejecutar un bloque de código repetidamente. Python ofrece dos tipos principales de bucles: `for` (para iterar sobre secuencias) y `while` (para repetir mientras una condición sea verdadera).

1. Bucle `for`

El bucle `for` se utiliza para iterar sobre una secuencia (como una lista, tupla, cadena o rango) o cualquier otro objeto iterable. Para cada elemento en la secuencia, el código dentro del bucle se ejecuta una vez.

2. Bucle `while`

El bucle `while` ejecuta un bloque de código repetidamente mientras una condición específica sea verdadera. Es importante asegurarse de que la condición eventualmente se vuelva falsa para evitar un bucle infinito.
