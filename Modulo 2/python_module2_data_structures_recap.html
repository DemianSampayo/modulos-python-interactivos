Repaso: Listas, Tuplas, Conjuntos, Diccionarios

Antes de sumergirnos en bucles y manejo de errores, es crucial tener frescos los conceptos de las estructuras de datos fundamentales de Python. Aquí te recordamos sus características principales.

Listas `[]`

Colecciones ordenadas, mutables, permiten duplicados. Acceso por índice.

mi_lista = [1, "dos", 3]

Tuplas `()`

Colecciones ordenadas, inmutables, permiten duplicados. Acceso por índice.

mi_tupla = (1, "dos", 3)

Conjuntos `{}` / `set()`

Colecciones desordenadas, mutables, NO permiten duplicados. Sin acceso por índice.

mi_conjunto = {1, 2, 2, 3} # -> {1, 2, 3}

Diccionarios `{}`

Pares clave-valor, mutables, claves únicas. Ordenados (desde Python 3.7).

mi_dict = {"a": 1, "b": 2}
